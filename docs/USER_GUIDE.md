## 1. Introduction

The **Anamnese-App** is a medical anamnesis (patient history) collection tool designed for healthcare professionals. It allows you to:

- Collect patient information securely
- Guide patients through structured questionnaires
- Export data in GDPR-compliant formats
- Work in 19 different languages

This guide will walk you through every feature step-by-step.

## 10. Privacy & Data Protection

### Your Data is Protected

The Anamnese-App is designed with **GDPR compliance** in mind:

| Feature | Protection |
|---------|------------|
| **Local Storage** | All patient data stays on your device |
| **Encryption** | Data is encrypted at rest |
| **No Cloud Upload** | Data never leaves your computer automatically |
| **Right to Deletion** | You can delete any patient's data completely |

### Deleting All Data

If you need to remove all data from the app:

1. Go to **Settings**
2. Scroll down to **"Danger Zone"**
3. Click **"Delete All Data"**
4. Type "DELETE" to confirm
5. Click **"Confirm Delete"**

## 11. Troubleshooting

### Problem: App Won't Start

**Solution:**
1. Restart your computer
2. Try running the app as Administrator:
   - Right-click on the app icon
   - Select **"Run as administrator"**

> **[TODO: INSERT SCREENSHOT HERE - SHOWING: Right-click context menu with "Run as administrator"]**

### Problem: Language Not Changing

**Solution:**
1. Go to **Settings**
2. Select your language
3. Close and reopen the app

### Problem: Cannot Save Patient

**Solution:**
1. Make sure all required fields are filled
2. Check that the date of birth is valid
3. Try closing and reopening the app

### Problem: Export Not Working

**Solution:**
1. Make sure you have permission to save to the selected folder
2. Try saving to your Documents folder instead
3. Check that you have enough disk space
